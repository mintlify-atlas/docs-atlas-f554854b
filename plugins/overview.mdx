## Plugin management

nvim-luffy uses [lazy.nvim](https://github.com/folke/lazy.nvim) for plugin management, providing:

- Fast startup times with lazy loading
- Automatic plugin installation
- Easy plugin updates
- Minimal configuration overhead

<Note>
  All plugins are configured in `lua/leyland/plugins/` with lazy loading enabled where appropriate to maintain fast startup times.
</Note>

## Installed plugins

Here's the complete list of plugins by category:

### Development

- **nvim-lspconfig** - LSP client configurations
- **mason.nvim** - LSP server installer
- **mason-lspconfig.nvim** - Bridge between Mason and lspconfig
- **mason-tool-installer.nvim** - Automatic tool installation
- **nvim-cmp** - Completion engine
- **LuaSnip** - Snippet engine
- **conform.nvim** - Code formatting
- **nvim-treesitter** - Syntax highlighting

### Navigation

- **telescope.nvim** - Fuzzy finder
- **nvim-tree.lua** - File explorer
- **harpoon** - Quick file navigation
- **flash.nvim** - Enhanced motion

### Git

- **neogit** - Git interface
- **diffview.nvim** - Git diff viewer

### UI

- **alpha-nvim** - Dashboard
- **snacks.nvim** - Smooth scrolling
- **nvim-web-devicons** - File icons
- **lspkind.nvim** - VS Code-like pictograms
- **trouble.nvim** - Diagnostics list

### Colorschemes

- **catppuccin** - Pastel theme
- **tokyonight.nvim** - Dark theme
- **rose-pine** - Minimal theme
- **bamboo.nvim** - Natural theme (default)
- **modus-themes.nvim** - Accessible themes
- **onedark.nvim** - Atom-inspired theme

### Utilities

- **plenary.nvim** - Lua utility functions
- **nvim-autopairs** - Auto-close brackets
- **todo-comments.nvim** - Highlight TODO comments
- **vim-tmux-navigator** - Tmux integration
- **lazydev.nvim** - Lua development tools
