## Welcome aboard the Thousand Sunny

Maybe the One Piece was the commits you made along the way. **nvim-luffy** is a fully-featured Neovim configuration that combines powerful development tools with the adventurous spirit of One Piece.

Built on top of [lazy.nvim](https://github.com/folke/lazy.nvim), this configuration provides a complete IDE experience right out of the box, with carefully selected plugins and keybindings optimized for modern software development.

## What makes nvim-luffy special?

<AccordionGroup>
  <Accordion title="Zero-configuration LSP setup">
    Language servers, formatters, and linters are automatically installed and configured. Just open a file and start coding with full IDE features including:
    - Intelligent code completion
    - Go-to-definition and find references
    - Inline diagnostics and error checking
    - Auto-formatting on save
  </Accordion>

  <Accordion title="Thoughtful keybindings">
    All keybindings use the space bar as the leader key, making them easy to remember and comfortable to use. Core features are accessible within two keystrokes:
    - `<Space><Space>` to find files
    - `<Space>/` to search text
    - `<Space>e` to toggle file explorer
    - `<Space>h` for Harpoon quick menu
  </Accordion>

  <Accordion title="Minimalist aesthetic">
    The configuration embraces a clean, distraction-free interface:
    - No status line clutter (hidden by default)
    - Winbar shows file info only
    - Alpha dashboard greets you with One Piece ASCII art
    - Clean colorscheme with the Bamboo theme as default
  </Accordion>

  <Accordion title="Fast and responsive">
    Built with performance in mind:
    - Lazy-loading plugins reduce startup time
    - Smooth scrolling with Snacks.nvim
    - FZF-native for blazing fast searches
    - No unnecessary bloat
  </Accordion>
</AccordionGroup>
